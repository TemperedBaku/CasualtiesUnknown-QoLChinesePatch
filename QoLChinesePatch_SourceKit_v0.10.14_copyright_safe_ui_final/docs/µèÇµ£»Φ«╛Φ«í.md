# v0.9.2 技术设计

v0.9.0 证明：精确克隆 `PreRunScript.instance/Button (11)` 可触发稳定汉化，但会在主菜单留下红色测试按钮。

v0.9.2 的目标是验证：触发器是否可以“存在但不可见”。

实现策略：

1. 仍然等待 `PreRunScript.instance`。
2. 找到 `Button (11)` 后克隆。
3. 按真实 disabled handler 写入 tooltip / image / button 结构。
4. 不销毁对象，保持 active。
5. 通过 CanvasGroup、Graphic、Selectable 与 RectTransform 将其隐藏并禁用交互。

如果本版仍能汉化，则后续可作为正式实现基础；如果失效，则说明触发器可能需要可见/可交互生命周期或额外 UI 事件。
